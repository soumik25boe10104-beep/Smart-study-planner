# 📘 Smart Study Planner (Python CLI Project)

## 📌 Description
Smart Study Planner is a simple command-line based Python application that helps students manage their study tasks efficiently. It allows users to add, view, complete, and delete tasks with priority levels.

This project is developed as part of the **Python Essentials BYOP (Bring Your Own Project)**.

---

## 🚀 Features

- ✅ Add new tasks  
- 📋 View all tasks  
- ✔ Mark tasks as completed  
- ❌ Delete tasks  
- ⭐ Set task priority (High / Medium / Low)  
- 💾 Data stored using JSON (persistent storage)  

---

## 🛠️ Technologies Used

- Python 3  
- JSON (for file handling)  
- CLI (Command Line Interface)
